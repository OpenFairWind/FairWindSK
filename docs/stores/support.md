# FairWindSK support

FairWindSK documentation and troubleshooting guides are available in the
[project documentation](../README.md).

To report a reproducible problem or request an enhancement, open an issue in the
[FairWindSK issue tracker](https://github.com/OpenFairWind/FairWindSK/issues).
Do not post passwords, vessel access tokens, precise private vessel positions,
or other sensitive information in a public issue.

When reporting a problem, include the FairWindSK version, operating system,
device model, Signal K server version, relevant logs with secrets removed, and
clear reproduction steps.
