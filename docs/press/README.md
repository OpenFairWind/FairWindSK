# FairWindSK press kit

This directory contains publication-ready press, social-media and boat-show material in every language supported by FairWindSK.

## Languages

- [`english/`](english/)
- [`french/`](french/)
- [`spanish/`](spanish/)
- [`italian/`](italian/)

Each language directory contains:

- press releases for boat magazines, yacht clubs and academia;
- ready-to-use social copy;
- 1080 x 1080, 1200 x 630 and 1080 x 1920 social-media images;
- a two-page A4 PDF brochure for printing and digital sharing.

All promotional figures use real FairWindSK interface captures from the repository. The localized social assets and brochures are generated by [`generate_press_kit.py`](generate_press_kit.py).

## Regeneration

From the repository root, run:

```bash
python3 docs/press/generate_press_kit.py
```

Before public distribution, confirm the release date, project contact details and current platform status.
